### Linux:

1. install nodejs and npm 
2. run build.sh 
3. if you get an error with electron try running `sudo npm install -g` 
- if you have any issues @ThemInspectors will try to help.

## Features

- Develop Mods in Lua language
- Syntax highligtning
- Mod-Specific projects
- DST API
- ModInfo Editor
- Integrated Steam-Workshop (search, downloading and uploading Mods)
- Asset Editor
- Console/Logger
- And lots of more...
